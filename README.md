# Система оценки показателей надежности и поддержки эксплуатации ИСБ
Разрабатываемая система предназначенна для автоматизации процессов сбора, хранения, обработки и анализа данных о техническом состоянии и надежности компонентов интегрированной системы безопасности. Основным целевым назначением системы является информационная поддержка принятия решений по техническому обслуживанию, ремонту и модернизации оборудования ИСБ с целью обеспечения требуемого уровня готовности системы и минимизации рисков, связанных с отказами.
